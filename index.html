My website

  1. First, I registered a domain name for my website using AWS Route 53. This allowed me to create a custom URL for my website and to manage its DNS settings.
  2. Next, I created an S3 bucket to store the static content for my website, such as HTML, CSS, and JavaScript files. I configured the bucket to host a static website and set up appropriate permissions to allow users to access the website.
  3. I then used CloudFront to create a content delivery network (CDN) for my website. This allowed me to distribute the content of my website to multiple edge locations around the world, providing faster access for users and reducing the load on the S3 bucket.
  4. To make the website more secure and reliable, I used AWS Certificate Manager to create a free SSL/TLS certificate for my domain. This allowed me to use HTTPS to encrypt traffic to and from the website, ensuring that the website and its users are protected from man-in-the-middle attacks.
  5. Finally, I setup GitHub Action script to monitor git repository for changes, if it comes to update script automatically update S3 bucket.
